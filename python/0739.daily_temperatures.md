# 739. Daily Temperatures

```python
def dailyTemperatures(temperatures: List[int]) -> List[int]:
    n = len(temperatures)
    result = [0] * n
    stack = []  # indices of colder days; stack kept monotonic decreasing
    
    for i in range(n):
        # If today is warmer than the last colder day,
        # resolve that past day: result[past] = i - past
        while stack and temperatures[i] > temperatures[stack[-1]]:
            past_day = stack.pop()
            result[past_day] = i - past_day
        # Only colder days are pushed to preserve monotonic decreasing order
        stack.append(i)
    
    return result
```
## Explanation of the Monotonic Stack Concept in This Problem

Let’s walk through `[73,74,75,71,69,72,76,73]` step by step with a **stack visualization**. This will give you a clear picture of how the monotonic decreasing stack works and why we only push colder days.

---

### Example Walkthrough

Temperatures: `[73,74,75,71,69,72,76,73]`  
We’ll track:
- **i** = current day index  
- **temp[i]** = today’s temperature  
- **stack** = indices of colder days waiting for a warmer one  
- **result** = days to wait for warmer temp

```text
Day | Temp | Action                          | Stack
---------------------------------------------------------
0   |  73  | push                            | [0]
1   |  74  | pop 0 → res[0]=1, push          | [1]
2   |  75  | pop 1 → res[1]=1, push          | [2]
3   |  71  | push                            | [2,3]
4   |  69  | push                            | [2,3,4]
5   |  72  | pop 4→1, pop 3→2, push          | [2,5]
6   |  76  | pop 5→1, pop 2→4, push          | [6]
7   |  73  | push                            | [6,7]

```
---


- **Stack holds colder days** → monotonic decreasing order.  
- **Condition**: `temperatures[i] > temperatures[stack[-1]]`  
  - If today is warmer, pop until monotonicity is restored.  
  - Each pop resolves a past day’s wait.  
- **Why decreasing?** Because we’re searching for the *next warmer day*. Storing colder days ensures the first hotter day resolves them immediately.  
- **Result meaning**: For each day, how many days until a warmer temperature.
